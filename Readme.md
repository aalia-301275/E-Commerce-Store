# E-Commerce Store

## 📌 Project Description
A simple full-stack e-commerce web application with product listing, product details, cart system, and checkout functionality.

## 🚀 Features
- Display products dynamically
- Product detail page
- Add to cart system (localStorage)
- Cart page with total calculation
- Checkout page with order confirmation alert
- Node.js backend API

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express

## 📁 Project Structure
Frontend → UI pages  
Backend → API server

## 🎯 How to Run
1. Open Backend folder
2. Run:# E-Commerce Store
